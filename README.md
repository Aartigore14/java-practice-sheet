## 🚀 How to Run

**Prerequisites:** Java JDK 17+ installed

```bash
# Clone the repository
git clone https://github.com/Aartigore14/java-practice-sheet.git

# Navigate to the project
cd java-practice-sheet

# Compile any file
javac Arrays/TwoSum.java

# Run it
java Arrays.TwoSum
```

Or simply open the project in **IntelliJ IDEA** and run any file directly.
